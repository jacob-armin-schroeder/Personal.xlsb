# Keyboard Shortcut Quick Reference

All shortcuts are assigned automatically by the installer, or manually via **Alt+F8 ‚Üí select macro ‚Üí Options**.

> `Ctrl+Shift+Letter` shortcuts require entering the **uppercase** letter in the shortcut field.

---

## AutoFill

| Shortcut | Macro | Description |
|---|---|---|
| `Ctrl+D` | `AutoFill` | Fill active cell down using adjacent column as length guide |

---

## Cell Formatting

| Shortcut | Macro | Description |
|---|---|---|
| `Ctrl+Shift+H` | `Border_Table_Heading` | Heading style: bold, centered, wrap text, vertical lines, outer border |
| `Ctrl+E` | `VerticalLines` | Cycle inside vertical borders: none ‚Üí thin ‚Üí medium ‚Üí none |
| `Ctrl+Shift+E` | `HorizontalLines` | Cycle inside horizontal borders: none ‚Üí hairline ‚Üí thin ‚Üí medium ‚Üí none |
| `Ctrl+Shift+O` | `Border_Outline` | Cycle outer border: none ‚Üí thin ‚Üí medium ‚Üí none |
| `Ctrl+Shift+N` | `Clear_Formatting` | Remove all borders and fill; reset font color |
| `Ctrl+Shift+B` | `FillBright` | Cycle light fill: none ‚Üí gray ‚Üí blue ‚Üí orange ‚Üí green ‚Üí white ‚Üí none |
| `Ctrl+Shift+D` | `FillDark` | Cycle dark fill (white font): none ‚Üí gray ‚Üí blue ‚Üí orange ‚Üí green ‚Üí black ‚Üí none |

---

## Find Changes

| Shortcut | Macro | Description |
|---|---|---|
| `Ctrl+M` | `FindNextChange` | Jump down to next different value in column |
| `Ctrl+Shift+M` | `FindPrevChange` | Jump up to previous different value in column |

---

## Number Formats

| Shortcut | Macro | Description |
|---|---|---|
| `Ctrl+Shift+A` | `NumberFormatDecimal` | Cycle decimal places: 0 ‚Üí 1 ‚Üí 2 ‚Üí 3 ‚Üí 4 ‚Üí 5 ‚Üí 6 ‚Üí (repeat) |
| `Ctrl+Shift+P` | `NumberFormatPercentage` | Cycle percentage decimal places: 0 ‚Üí 1 ‚Üí 2 ‚Üí 3 ‚Üí (repeat) |
| `Ctrl+Shift+C` | `NumberFormatCurrency` | Cycle currency formats: simple ‚Üí red-negative ‚Üí accounting ‚Üí (repeat) |
| `Ctrl+Shift+T` | `NumberFormatDateTime` | Cycle date/time formats; autofits column |

---

## Row and Column Size

| Shortcut | Macro | Description |
|---|---|---|
| `Ctrl+Shift+W` | `Autofit` | Autofit column width and row height |
| `Ctrl+Q` | `ColumnWidthIncrease` | Increase column width by 1 unit |
| `Ctrl+Shift+Q` | `ColumnWidthDecrease` | Decrease column width by 1 unit |
| `Ctrl+R` | `RowHeightIncrease` | Increase row height by 5 units |
| `Ctrl+Shift+R` | `RowHeightDecrease` | Decrease row height by 5 units |
